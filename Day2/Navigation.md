Навигация и работа с файлами
Базово в работе DevOps нужна для работы с сервером
Поиск файлов, написание скриптов, и общая работа с сервером

команда find
осуществляет поиск на основе парамеров пользователя

структура
find [путь] [параметры] [выражение]

параметры
по имени
-name "имя" - поиск по точному имени с учетом регистра
-iname "имя" - поиск без учета регистра

по типу объекста
-type "f обычный файл, d директория, l символическая ссылка"

по размеру
<F8>  -size

с -  байты
k - килобайты
M - мегабайты
G - гигабайты
b - байтовые блоки

биты не ищет минимум байт ( с )

по времени
есть только дни и минуты
-ntime 1 - 24 часа назад
-ntime +30 - более 30 дней назад

-nmin -60 - менее часа назад
-nmin +120 - более двух часов назад

-time отбрасывает дробную часть, если файл изменен 1.9 дня назад для системы он будет как 1 день назад 


команда locate
структура 
locate [параметры] []

-i поиск без учета регистра
-с количество файлов подходящих под параметры 
-n [number] вывод n файлов подходящих под параметры
-r выполняет поиск по особому шаблону
-e проверяет наличие файла а не только его ссылки (файл удален ссылка осталась)

команда file

команда stat

шпора
ПОИСК:
find — поиск файлов
locate — быстрый поиск (база)
grep — поиск по содержимому

ФАЙЛЫ:
cp — копировать
mv — перемещать
rm — удалять

ПРОСМОТР:
less — просмотр
tail -f — логи в реальном времени
head — начало файла

ПРАВА:
chmod — права
chown — владелец/группа

ДИСК:
du — размер
df — свободное место

ИНФО:
file — тип файла
stat — метаданные

СТРУКТУРА:
tree — дерево каталогов

PIPELINE:
| — передача вывода
xargs — превращает вывод в аргументы

ДОП:
wc — считать строки
sort / uniq — обработка данных
